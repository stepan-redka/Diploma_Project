Information Security Policy for TechCorp Inc.

1. Password Requirements
All employees must use passwords with at least 12 characters, including uppercase letters, lowercase letters, numbers, and special symbols. Passwords must be changed every 90 days. Reusing any of your last 5 passwords is prohibited.

2. Remote Work Guidelines
Employees working remotely must connect through the company VPN at all times. Personal devices may only access company email through the approved mobile app. Downloading sensitive documents to personal devices is strictly forbidden.

3. Data Classification
Data is classified into three levels: Public, Internal, and Confidential. Public data can be shared freely. Internal data should only be shared within the organization. Confidential data requires encryption and manager approval before sharing.

4. Incident Reporting
Any security incidents, including phishing attempts, lost devices, or suspicious emails, must be reported to the IT Security team within 24 hours. The security hotline is available 24/7 at extension 5555.

5. Software Installation
Only IT-approved software may be installed on company devices. Requests for new software must be submitted through the IT portal and require department head approval. Unauthorized software will be automatically removed during weekly security scans.
